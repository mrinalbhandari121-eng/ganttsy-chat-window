Free is usually best so I recommend starting there first. If we need to, we can explore the best fit and price if you have to make any purchases.
1Start with existing network first⌄

Most people ask strangers first when they should ask acquaintances. Many people have $500–$1,000 worth of tools gathering dust in a garage.

Make a list of:

  • Friends
  • Family
  • Neighbours
  • Co-workers
  • People from clubs or organizations you're involved with

Send a simple message:

Hi everyone, I'm building a small shed over the next couple of weekends and I'm hoping to borrow a cordless drill and circular saw rather than buying tools I'll only use briefly. If anyone has either available, I'd be happy to pick them up, treat them carefully, and return them promptly. Thanks!
2Facebook community groups⌄

Search words to try: "tool library", "borrow drill", "loaner saw", "Buy Nothing [your neighborhood]", "[your city] makerspace".

Hey neighbours — building a backyard shed this month and looking to borrow a cordless drill + circular saw for a weekend. Happy to drop something off in return (cookies, beer, or yard help). DM me!
3Check local Tool Libraries / community orgs⌄

Many cities run actual tool libraries — you check tools out like books. Conduct a local search using "[your city] tool library", "makerspace", or local Repair Café listings.

4Outside the box thinking… find a retired woodworker⌄

Many retirees:

  • Have fully equipped workshops.
  • Like seeing someone actually build something.
  • Are often happy to lend tools or even help.

A surprising number will say:

"Come over Saturday and use mine."

Sometimes renting is actually a better choice than free. Here's why:

1Reliability and Availability⌄

Your project schedule depends on you, not someone else's garage.

With borrowed tools:

  • "Can I keep it another day?"
  • "I need it back tonight."
  • Battery is dead.
  • Charger is missing.

With rentals:

  • You know exactly when you'll have the tool.
  • You know exactly when it must be returned.
  • Your shed build doesn't stall because Bob's grandson suddenly needs his drill.
2Better Tools = Faster Project⌄

The average homeowner's drill may be:

  • 10 years old
  • Weak battery
  • Dull blade
  • Missing accessories

Rental tools are often commercial-grade. A good circular saw can cut lumber noticeably faster and cleaner than an old homeowner model. For a shed, that can save several hours.

3No Relationship Risk⌄

Borrowing introduces social debt. If you:

  • Drop the tool
  • Burn out the motor
  • Lose the battery
  • Forget to return it

… you've potentially damaged a relationship. Renting turns the arrangement into a simple transaction.

4You Can Get Exactly What You Need⌄

A neighbour may have:

  • A hand saw instead of a circular saw.
  • A drill but no driver bits.
  • A tiny cordless drill that struggles with deck screws.

Rental shops let you select:

  • Circular saw
  • Mitre saw
  • Impact driver
  • Air nailer
  • Compressor

… matching the actual job.

5Hidden Cost of "Free"⌄

Free isn't always free. You might spend:

  • 3 hours locating tools.
  • Multiple phone calls.
  • Driving around town.
  • Waiting for someone to be available.

If a drill rental costs $20–30 for a day and saves half a day of hunting, it can be the cheaper option overall.

1What to budget for⌄

Entry-level cordless circular saw (homeowner-grade): $80–$140 — fine for one shed build, may struggle with hardwood.

Mid-range (recommended): $180–$280 — 18V/20V, 7¼" blade, ~4Ah battery, will last decades for occasional use.

Pro-grade: $320+ — brushless motor, multiple batteries, faster cuts. Worth it only if you'll use it monthly.
